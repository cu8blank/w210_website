Currency Prediction

We analyze currency exchange rate data from the International Monetary Fund (IMF). By applying an ARIMA model to each dataset, we forecast Special Drawing Right (SDR) data for all currencies tracked by the IMF. We provide the user with 7-day, 30-day, and 90-day forecasts.

Country Information Prediction

Using data from Tripadvisor and Wikipedia's Visa Requirements for US Tourists to filter destination countries based on user preferences. The data from Tripadvisor helps us to create topics on all the countries user review comments. Additionally, we performed another unsupervised learning algorithm to cluster similar countries together based on descriptive attributes. Finally, the average attraction rating for the country was calculated.
